# Face Recognition Project

This is a simple Face Recognition system built using Python and OpenCV. It detects and recognizes human faces in real-time using a webcam. This project was created as part of the PH4E subject requirement.

## Features
- Real-time face detection using webcam
- Face recognition of known individuals based on trained data
- Easy to use and lightweight

## Technologies Used
- Python
- OpenCV
- NumPy
